💼 Personal Business Card Website

This is a simple personal business card website created using HTML and CSS.  
It presents basic information about me, my goals, and ways to get in touch.

📌 About the Project

The website includes:
- A profile photo
- My name and role
- Current goal (seeking Ausbildung)
- Contact information
- Links to portfolio and GitHub

The design is clean and minimal, focused on readability and simplicity.

🚀 Technologies Used

- HTML5
- CSS3

🎯 Purpose

This project was created as part of my learning journey in web development.  
It helps me practice:
- Structuring a webpage
- Styling with CSS
- Creating a simple and professional layout
